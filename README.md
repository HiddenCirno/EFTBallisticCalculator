# 星火计划·火控系统

为SPT所设计的火控和信息UI模组。

**它能做什么？** 

- 在你使用瞄具的时候根据给定的距离，预测当前的弹道落点。

- 显示包括距离，方向，角度，子弹参数，预计子弹初速，飞行时间等数据。

- 显示当前玩家的健康状态，以及简易的队友状态（WIP）。

- 显示一些很Cool~的东西在屏幕上，比如风速，风向，气温，气压，当然，它们全部是假的：P

**如何使用？** 

- 在右侧的Release下载发布版本，解压后得到BepInEx文件夹，直接覆盖于游戏根目录即可。

**兼容性如何？**

- **与Fika的兼容性：** 完全兼容，我在实际测试中没有遇到任何问题。

- **与Fika.Co-op的兼容性：** 不确定，尚未测试，我也没有测试的条件，火控系统依赖于客户端的Camera系统实现，但愿BSG没有渲染其它人的Cam。

- **与Fika.Headless的兼容性：** 不确定，尚未测试，同上。

- **与Realism的兼容性：** 不确定，但我觉得大概率不兼容，这个项目构建于4.0.13版本，而Realism目前停留在3.11.4版本，我抽取了弹道算法并重建为工具方法，但是Realism应该有自己的弹道参数。

**关于用法**

- 进入游戏后，你的屏幕左侧会出现两个信息面板，分别是火控系统的状态和环境传感器的信息（除时间和地点外的所有环境信息都是模拟的）。
- 火控系统会显示当前的距离、俯仰角、水平倾角、子弹参数、离膛速度、飞行时间等数据。
- 按下T键将准星位置设置为目标点，火控系统将会自动测量你与落点的水平距离，并计算火控数据和进行落点预测。
- 预测落点表现为一个黄色的发光点，它会在开镜状态下显示。
- 如果准星瞄准的位置无法成为目标（如天空），将会清除距离数据。
- 按下Backspace键也可以手动清除火控数据。
- 通过Ctrl、Alt或Shift键和上下方向键，可以手动输入距离数据或对当前数据进行校准修正。
